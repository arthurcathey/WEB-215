Proof of Concept 3: DOM

Recall that methods are essentially functions that are built in to an object. Methods, as they are functions, require parenthesis after the method name and sometimes require arguments. Properties, unlike methods, do not require parenthesis and are set via an equal sign.

Elements can be accessed via properties or methods. Elements can also be manipulated via methods or properties.

Some Helpful Methods

Selecting Elements
  • getElementByClassName()
  • getElementById()
  • getElementsByTagName()
  • querySelector()
  • querySelectorAll()

Manipulating CSS Classes

  • add()
  • remove()
  • toggle()

Creating and Inserting Elements

  • appendChild()
  • createElement()
  • insertAdjacentElement()
  • replaceWith()

Inserting Text

  • insertAdjacentText()
